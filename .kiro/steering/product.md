# Product Overview

marry.fun は恋愛シミュレーションゲーム（ギャルゲー風）の Web アプリケーションです。3D キャラクターとチャットによる没入感ある体験を提供し、将来的なサーバー連携やトークン報酬を想定した設計になっています。

## Core Capabilities

- **LP 体験**: BGM ループ再生、ロゴアニメーション、3D モデル表示によるファーストビュー
- **チャット**: キャラクターとの会話シミュレーション（ポイント・感情表現付き）
- **国際化**: 日本語・英語の二言語対応
- **レスポンシブ**: モバイル/デスクトップ双方での安定した表示

## Target Use Cases

1. 新規訪問ユーザーが LP で marry.fun の世界観を短時間で把握する
2. ユーザーがキャラクターとチャットし、ポイント獲得・目標達成を楽しむ
3. 多言語ユーザーが好みのロケールで体験する

## Value Proposition

ギャルゲー風のビジュアル（パステル/ポップ、3D 人形、BGM）とチャットゲームプレイを組み合わせた Web 体験。アクセシビリティ（prefers-reduced-motion 対応）を考慮しつつ、没入感を提供する。
