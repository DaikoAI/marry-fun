# Product Overview

marry.fun は OpenClaw Hackathon 向けに作られた AI 恋愛シミュレーションゲームです。「Claw-chan」という AI ヒロイン（Moltworker + OpenClaw）とチャットし、スコアを競って本物の報酬を目指します。

## Core Capabilities

- **LP 体験**: BGM ループ再生、ロゴアニメーション、3D モデル表示によるファーストビュー
- **チャット**: Claw-chan との会話（5 タイプの隠れた性格、ポイント・感情表現）
- **NG ワード**: セッション開始時に 20 語の禁止ワードを秘密生成。発言で即 Game Over
- **$MARRY トークン**: ホルダーボーナスでスコア倍率アップ
- **国際化**: 日本語・英語の二言語対応
- **レスポンシブ**: モバイル/デスクトップ双方での安定した表示

## Target Use Cases

1. 新規訪問ユーザーが LP で marry.fun の世界観を短時間で把握する
2. ユーザーが Claw-chan とチャットし、隠れた性格に合わせてポイント獲得・目標達成を楽しむ
3. $MARRY トークンホルダーがボーナスを活かしてスコアを競う
4. 多言語ユーザーが好みのロケールで体験する

## Value Proposition

ギャルゲー風のビジュアル（パステル/ポップ、3D 人形、BGM）とチャットゲームプレイを組み合わせた Web 体験。アクセシビリティ（prefers-reduced-motion 対応）を考慮しつつ、没入感を提供する。日次トッププレイヤー・最終日のマリッジパートナーには $MARRY creator 手数料の分配が行われる。
